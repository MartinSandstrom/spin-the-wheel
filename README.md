# Spin the wheel


## Preview 
https://martinsandstrom.github.io/spin-the-wheel/


## Usage
    
### Clone

```console
    $ git clone https://github.com/MartinSandstrom/spin-the-wheel.git
```

```console
    $ cd spin-the-wheel
```

### Install

```console
    $ npm install
```

### Run dev server

```console
    $ npm start
```
